Размер иконки должен быть   40х40px .ico

***для встроенной иконки***
1. Скопировать иконку в папку Res\Resources.  
   Дать имя с номером.
   Зарезервированные имена: 70 - стандартная иконка путевой точки
							72 - начало маршрута
							73 - конец маршрута
							74 - стандартная иконка "Что здесь?"
2. Добавить ее в ресурсы проекта Res
3. Добавить в метод TrackConverter.Res.IconOffsets.GetOffset(int icon) сдвиги по осям для этой иконки
   Сдвиг добавлять в соответствующий раздел метода (частный случай, диапазон)

***для иконки из файлов Images/icons
1. иконка формата .ico скопировать в папку Images/icons дать имя по образцу (смотреть заразирвированные имена выше)
2. добавить в проект и поставить свойство "копировать в выходной каталог"
3. Добавить в метод TrackConverter.Res.IconOffsets.GetOffset(int icon) сдвиги по осям для этой иконки
   Сдвиг добавлять в соответствующий раздел метода (частный случай, диапазон)
