TrackConverter

Программа предназначена для создания веломаршрутов с использованием карт Google, Яндекс и OSM CycleMap.
Есть возможность получения высоты точки над уровнем моря (используются разные сервисы, такие как Google, GTOPO30, ETOPO2).
Экспорт карт с привязками в файлы изображений JPEG и BMP. В будущем планируется поддержка тайловых форматов ECW и OZF.
Работает с фалами GPS треков, файлами OZI Explorer, Google Earth и некоторыми другими форматами.
Полный список поддерживаемых форматов: rt2, crd, plt, wpt, kml, kmz , rte, gpx, csv, osm, nmea, txt, adrs.
Помимо файловых форматов поддерживается работа со ссылками на маршруты Яндекс-карт и сайта Wikimapia.org.
Для удобства можно сокращать ссылки с помощью сервисов Clck.ru и Qps.ru.

Инструменты

  • Оптимальный маршрут через заданные точки: по прямой или с использованием сервисов прокладки маршрутов: Google, Yandex;
  • Инструмент сравнения маршрутов по параметрам: длина, перепады высот, максимальные отклонения по высоте, длина подъемов и спусков и т. д;
  • Инструмент перевода географических координат в различные форматы: долгота, широта, десятичные значения градусов;
  • Редактирование путевых точек, работа с точками на карте;
  • Построение профиля высот маршрута;
  • Экспорт изображений карт в файлы JPEG и BMP с привязками.
